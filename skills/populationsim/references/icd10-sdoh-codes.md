# ICD-10-CM SDOH Z-Codes Reference

## Overview

ICD-10-CM includes Z-codes (Z55-Z65) to document social determinants of health affecting patient care. These codes support SDOH screening, care coordination, and population health analytics.

**Code Range**: Z55-Z65 (Persons with potential health hazards related to socioeconomic and psychosocial circumstances)
**Effective**: ICD-10-CM 2024
**Updates**: Annual (October 1)

## Usage Guidelines

### Primary vs Secondary Diagnosis

| Use | Guidance |
|-----|----------|
| **Primary** | Only when SDOH is focus of encounter |
| **Secondary** | To document contributing factors |
| **Recommended** | Secondary diagnosis in most cases |

### Documentation Requirements

- Document specific SDOH factor
- Link to health impact if possible
- Update when circumstances change
- Screen systematically (e.g., PRAPARE, AHC)

### Screening Tools

| Tool | Source | Codes Mapped |
|------|--------|--------------|
| PRAPARE | NACHC | Z55-Z65 comprehensive |
| AHC HRSN | CMS | Core SDOH domains |
| WE CARE | Academic | Pediatric focus |
| IHELLP | Academic | Pregnancy focus |
